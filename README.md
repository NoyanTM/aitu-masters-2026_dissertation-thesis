# aitu-masters-2026_dissertation-thesis
Masters's degree dissertation thesis for "7M06106 Secure Software Engineering" educational program at Astana IT University 

## Description
- Based on "the University of Southampton Thesis Template" and Kazakhstan's GOST requirements (bibliography and format)
- It is one of the foundation for `codecore` which initiates related ecosystem of solutions/methods (`codestash`, `fastsandbox`, `codeless`, `codeload`, etc.)

## Requirements
- Used software: texlive, biber, gnu make, drawio
- /figures are built from diagrams.drawio
- Build latest thesis.pdf via `make build`
